title: "On building things that last"
date: 2026-03-26
categories: ["Ideas", "Building"]

---

The temptation is always to chase what's new.

New frameworks, new platforms, new trends. But the things that endure are usually built on boring foundations. Consistency. Simplicity. A clear purpose.

A blog doesn't need to be complicated. It needs to exist. It needs to be updated. It needs to say something worth saying.

The same is true for most things we build. The fancy parts are rarely the parts that matter.

What matters is: does it work? Does it serve its purpose? Will you still want to maintain it in a year?

If yes, you've built something that lasts.
